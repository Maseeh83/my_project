# Python Desktop Application

A Python desktop application built with wxPython that processes data from CSV files,
stores results in a local SQLite database, and displays them in a graphical interface.

## Features
- GUI built with wxPython
- CSV data processing using NumPy
- Local SQLite database storage
- Modular codebase (23 Python files)

## Technologies
- Python
- wxPython
- NumPy
- Matplotlib
- SQLite

## How to Run
```bash
python main.py
